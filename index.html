Game Projects

  • 3x-Eh (A Canadian 4X)


    I worked on this game with 7 others over a period of just under 4 months. It was done for a special course being offered to a group selected from a pool of applicants, for this year only (for Canada's 150).

  • Flipside

    Flipside is a small platformer created in Unity for the 2016 Global Game Jam. This is the first game I've ever worked on with proper art assets created for the game, and coupled with the previous work I did in Unity last year, it came out looking fantastic and playing quite well!

    GGJ Site
    Github Page Playable Web Version

  • BPM


    A small RTS made in Unity. Units have pathfinding, animations, and a few particle effects. A basic AI is also included, with pathfinding and target acquisition of it's own.

    Github Page

  • Heli


    A little helicopter simulator. Can slow down time and everything. There's also a jeep and a tank that will shoot at you. Physics calculations are done using SAT so that any convex polygon can be processed.
